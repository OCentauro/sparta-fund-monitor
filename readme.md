# Sparta Fund Monitor

Monitoramento de fundos imobiliários geridos pela Sparta Asset Management com dados em tempo real e indicadores fundamentais.

![Status](https://img.shields.io/badge/status-active-success)
![License](https://img.shields.io/badge/license-MIT-blue)
![GitHub Pages](https://img.shields.io/badge/deployed-GitHub%20Pages-orange)

## 🚀 App Online

**Acesse:** https://ocentauro.github.io/sparta-fund-monitor/

## 📊 Funcionalidades

- ✅ Preços em tempo real via BrAPI
- ✅ Indicadores fundamentais (P/VP, P/FFO, DY) atualizados mensalmente
- ✅ Links diretos para Google Finance e página oficial da Sparta
- ✅ Interface responsiva (PC e mobile)
- ✅ Dark mode automático
- ✅ Lembrete automático via Telegram (dia 11 de cada mês)

## 🛠️ Stack Tecnológica

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **API de Preços:** [BrAPI](https://brapi.dev)
- **Dados Fundamentais:** [Fundamentus](https://fundamentus.com.br) (web scraping)
- **Deploy:** GitHub Pages + GitHub Actions
- **Automação:** Python (BeautifulSoup, Requests)
- **Notificações:** Telegram Bot API
